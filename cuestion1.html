¿Qué tipado se aplica en JavaScript?

El tipado que se aplica en JavaScript es débil, ya que a la hora de declarar las variables no se les asigna el tipo de variable (String, int, double, boolean...) como en otros lenguajes fuertemente tipados como Java, por lo que se permite, por ejemplo, manipular una variable poniendo otro tipo de dato una vez creada dicha variable.
